# 컨벤션

## 좋은 컨벤션 (naver)

```
<!-- 여는 주석 -->
<element>
...
</element>
<!-- 닫는 주석 -->
```

---

## 주의할 요소

-   `iframe`은 해킹에 취약하다.

---

## 많이 쓰는 규칙(HTML, CSS)

-   대문자를 쓰지 않는다. (ex. ` #FFFFFF` X `#ffffff` O)
-   class 선택자 앞에 의미 없이 부모를 쓰지 않는다. (`element .selector` X `.selector` O)
